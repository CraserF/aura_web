# Copy Rules

- Write titles as claims, not labels.
- Keep kickers under 42 characters and make them slide-specific.
- Use footers for source, date, section, or navigation context.
- Do not duplicate kicker text in the footer.
- Use one dominant sentence per slide.
- Keep body copy concrete and avoid generic transformation language.
- Never invent metrics. If a number is not in source material, ask, cite, or remove it.
- Prefer verbs over abstract nouns.
- Avoid emoji and decorative icon substitutes.
- Write captions as evidence context, not image descriptions alone.
- For decisions, name the recommendation, risk, trade-off, and ask.
- For closing slides, include owner, next action, and checkpoint.
